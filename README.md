# 🎓 Academic Student Registry System

A full-stack student management application designed to organize student registration data. It enables administrators to create, read, update, and delete (CRUD) student records via a secure PHP backend linked to a MySQL database.

## 🚀 Features

- **Secure Login & Registration**: User access control with registration and login screens.
- **Live CRUD Interface**: Seamless create, read, update, and delete capabilities for managing student profiles.
- **Database Integration**: Powered by MySQL with a pre-configured database schema (`student.sql`).
- **Clean Styling**: Standardized CSS interface for modern dashboard layouts.

## 🛠️ Technologies Used

- **Backend**: PHP, MySQL
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: SQL

## 📂 Project Structure

- `conn.php` - Database connection utility
- `addUser.php` / `adddata.php` - Record creation logic
- `updatedata.php` / `deletedata.php` - Update and deletion endpoints
- `login.php` / `register.php` / `logout.php` - User authentication
- `student.sql` - Pre-populated database database schema
